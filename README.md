# БДУ ФСТЭК Анализатор

Десктопное приложение для анализа Банка данных угроз безопасности информации ФСТЭК России.

## Возможности

- Сводная панель — ключевые метрики, графики
- Вендоры / ПО — топ вендоров, скорость устранения
- Временной анализ — динамика по годам, тренд критичности
- Критичность — CVSS распределение, CWE тепловая карта
- Статусы — подтверждение, устранение, эксплойты
- БД / SQL — просмотр таблицы + произвольные SQL-запросы
- Экспорт — CSV, Excel, JSON на каждой странице

## Быстрый старт

pip install PyQt6 pandas matplotlib openpyxl
python main.py

## Сборка в .exe

pip install pyinstaller
pyinstaller fstec_analyzer.spec
